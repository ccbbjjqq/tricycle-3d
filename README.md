# 三轮车不锈钢框架 3D 示例图

这是一个用于查看夜市三轮车不锈钢框架的 3D 尺寸说明网页。

## 在线查看

打开下面的网址即可查看 3D 示例图：

https://ccbbjjqq.github.io/tricycle-3d/

## 使用说明

- 电脑端：鼠标拖拽旋转，滚轮缩放。
- 手机端：单指旋转，双指缩放。
- 页面内包含三轮车不锈钢框架的三维示意图、参数表、管材下料清单和材料汇总。

## 文件说明

- `index.html`：3D 示例图网页文件。
- `README.md`：当前说明文件。

## 注意事项

页面中的尺寸和材料清单仅作为加工沟通参考，实际制作前需要以实车复尺为准。焊接前建议先点焊定位确认，关键受力位置满焊，焊口打磨抛光。
